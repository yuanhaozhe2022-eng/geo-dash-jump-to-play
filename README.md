this is a simple project i made in 20 minutes for fun
basically what it does is that you have an accelerometer mounted to your body
everytime you jump, it detects the movement and presses space
HOW TO SET IT UP:
first, get an Arduino (nano perferred) and a MPU6050 module
you might also want a long usb for the arduino
then wire the MPU6050 to the arduino which you could just look up on the internet for how (It's I2C)
after that, flash the arduino sketch
lastly, get your arduino connected and run the python script, .
then everytime you jump, the pyautogui presses space and you only have to load up your geometry dash and that's it!
